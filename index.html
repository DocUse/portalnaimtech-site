NaimTech Recruiter Portal

Это внутреннее веб-приложение для сотрудников, которым предоставлен доступ к личному кабинету рекрутера.

Приложение показывает персональные рабочие данные сотрудника: текущие кейсы, дисциплинарные баллы, зоны и краткую сводку по его рабочим показателям.

Для чего используется приложение

Приложение нужно, чтобы сотрудник мог открыть свой личный кабинет и увидеть только свои данные, связанные с рабочими кейсами и дисциплиной.

Кто может получить доступ

Доступ открывается не всем автоматически. Администратор заранее вносит Google email сотрудника в служебную таблицу и включает доступ к порталу.

Какие данные используются

  • адрес Google email для определения личности пользователя;
  • данные из служебной Google Таблицы для отображения личного кабинета;
  • служебные настройки приложения.
